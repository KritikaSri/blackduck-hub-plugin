If the Project Name you have provided is unique, this list should be empty.
If the Project Name you have provided has multiple matches on the Hub server then this list will be populated with the Id's of the matching projects.
There is a 2 second delay so that the Project Name can be checked against the Hub server for matches before filling out this drop-down list.

If there is a warning that there are multiple Projects in the Hub with the same name but the "Duplicate Projects, Id List" is empty:
* Click away from the field, wait a second and the list should be filled.
* If the list is not populated after a second or two :
   * Change the Project Name
   * Click away from the field
   * Change the Project Name back to what you want it to be
   * Click away from the field
   * The list should be populated after a second or two

If there is a message saying that the project does not exist but the "Duplicate Projects, Id List" still contains the Id's of a Project that had multiple matches:
* Click away from the field, wait a second and the list emptied.
* If the list is not emptied after a second or two :
   * Change the Project Name
   * Click away from the field
   * Change the Project Name back to what you want it to be
   * Click away from the field
   * The list should be emptied after a second or two
